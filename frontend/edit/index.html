Class-Sync Timetable Manager

Admin Master Control Panel

Timetable Viewer

This portal is used to view timetable of the sections, Faculties and Rooms.

Timetable Editor

This portal is used to edit timetable of different sections. It is a read-write portal.

Generic Subject Data Setter

This page is used to enter subject schemes (subject data) of different courses.

Faculty Setter

This portal is used to set the faculty data for all sections across courses and semesters.

Faculty Data Editor

This page is used to add, remove and modify Faculty details. Faculties then can be assigned.

Room Editor and Allotter

This portal is used for allotment of rooms to the courses and to add, remove and edit rooms.

Reset Database

This portal is used to reset the timetable data. Warning: This action is irreversible.

Extra mix Features

All the miscellaneous and other features are available here.
